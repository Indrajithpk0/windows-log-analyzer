# üîç Windows Log Analyzer (Cybersecurity Project)

A real-time browser-based Windows Event Log Analyzer.  
Users can upload their `.csv` log file and visualize suspicious activities such as failed logins, privilege escalation, and audit policy changes.

### Features
- Detects Event IDs: 4625, 4672, 5379, 4907  
- Displays total and suspicious events  
- Interactive chart of failed login attempts by IP  

### How to Use
1. Go to [Project Website](https://<your-username>.github.io/windows-log-analyzer/)
2. Upload your Windows log file (converted to `.csv`)
3. View real-time analysis results instantly in your browser.

### Tech Stack
- HTML, CSS, JavaScript  
- PapaParse.js (CSV Parsing)  
- Chart.js (Visualization)  

---

**Cybersecurity Insight:**  
This project demonstrates log analysis and visualization techniques for detecting brute-force, credential access, and privilege misuse attempts on Windows systems.
